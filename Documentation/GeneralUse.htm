The program can be used in one of two ways.

1. Listen for pictures to be transmitted on the serial port and convert them as they com in
2. Convert a picture already captured from the serial port (e.g. using a terminal program)

To use the program in listen for pictures on the serial port mode:
1. Click the "Listen for scope data button on the tool bar"
2. Use the appropriate controls on the oscilloscope to initiate a transfer
3. Picture will be displayed after the transfer and conversion is complete. If the autosave flag (in the setup) is checked, the program will also save the picture.

To use the program in conversion mode:
1. Under file, click "Convert Raw File"
2. In the file selection box, find the transferred picture and click open. The picture will then be converted and displayed. If the autosave flag (in the setup) is checked, the program will also save the picture.
