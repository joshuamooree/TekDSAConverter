Welcome to my scope transfer program.

At this time, this program is intended to transfer and convert image files from the Tektronics DSA 602. It may also work with oscilloscopes in the Tek 11k series, but I have not tested this.

This program can
  • Transfer the picture file from the oscilloscope
  • Convert already transferred pictures from the oscilloscope
  • Save pictures in several different formats
  • Open and display picture files in several different formats
