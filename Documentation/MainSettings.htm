The main settings page has settings for the autosave and for the serial port setup.

At the top of the page is the settings for the serial port. Make sure these match the settings of the oscilloscope, otherwise the transfer will not occur.

At the bottom of the page is the settings for the autosave feature.
The checkbox labeled "Autosave Pictures" allows the auto save feature to be turned on and off.
The text box under Picture name format allows the name of the picture to be controlled. The name will be the text specified by the text box, followed by a number, followed by the type of the picture (controllable by the drop down menu). The format of the first part of the name may be changed by replacing the text. The name allows variable substitutions to be used. Valid substitutions are:
  • %% for the % character
  • %d for the current date
  • %t for the current time
  • %y for the current year
  • %M for the current month
  • %h for the current hour
  • %m for the current minute
  • %s for the current second
The text box labeled "Autosave location" allows the location of the saved files to be chosen. By default it is the current working directory ./ (usually the directory with the executable). Use the button labeled .. to pick a different directory or enter a different directory in the box.
